# Example capsules

Teaching artifacts for LoopCompass shape and lanes. **Not** live project memory.

- Do not copy these into a consumer `.loopcompass/` as if they were verified for that repo.
- Bodies are **domain-neutral** redactions of real pilot patterns (browser automation walls,
  extension load flags, sandbox caches, timezone identity bugs, worktree spawn, null keepalive).
- Live dogfood stays in private consumer repos; run `node scripts/redact-check.mjs examples` before
  publishing new examples.

Each file follows the recovery or incident template and passes `validate-state` rules.
